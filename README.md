# Voice AI Agent

## Description
This project is a simple AI agent that takes user input and performs tasks like:
- Creating files
- Writing code
- Summarizing text

## Tech Used
- Python
- Streamlit

## Features
- Intent detection
- File creation
- Code generation
- Text summarization

## Note
Used text input instead of audio due to system limitations.

## Run
pip install streamlit
python -m streamlit run app.py
